# Arduino House Device

Physical smart home implementation using Keyestudio KS0085 kit.

## Setup
1. Disconnect HM-10 module before uploading firmware
2. Upload Project_test.ino via Arduino IDE
3. Reconnect HM-10
4. pip install bleak requests
5. Run python bridge.py from SE_prototyp/

## Requirements
- Python 3.11+
- Laptop with BLE adapter, physically near the house
